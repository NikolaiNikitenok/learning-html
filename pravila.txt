селектор тэга слабее чем class

среди двух селекторов победит тот, у которого больше тэгов

среди классов победит тот, у которого указано больше классов

Селектор родителя слабее селектора потомка

class слабее  id

соседний селектор = +

родственный селектор = ~

> = непосредственно внутри тега

атрибут в [] = a [href] {}
Атрибут с началом = a [href^ = ''] {}
Атрибут с концом = a [href$ = ''] {}
Атрибут в котором встречается определенный текст = a [href* = ''] {}
Стиль применяется к элементам, у которых атрибут начинается с указанного 
значения или с фрагмента значения, после которого идёт дефис = div [class|="block"] {}
Стиль применяется в том случае, если у атрибута имеется указанное значение 
или оно входит в список значений, разделяемых пробелом = [class~="block"] h3 { color: green; }

border: dotted, dasher, ridge, double, solid

inline-flex подгоняет родителя по ширене под потомка

for flex use justify-content

position: absolute; становится невидимкой для других объектов

position: relative; Смещение относительно начальных координат

Если элементу задан relative, а его потомку - absolute, то этот потомок будет позиционироваться относительно своего родителя, а не относительно окна браузера.
