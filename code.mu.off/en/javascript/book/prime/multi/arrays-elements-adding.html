Adding elements to multidimensional arrays in JavaScript

Suppose we have such an array with users:

let users = [ { name: 'name1', surn: 'surn1', }, { name: 'name2', surn: 'surn2', }, { name: 'name3', surn: 'surn3', }, ];

Let's add one more user to this array. To do this, as a parameter of the push method, we will pass a new object with a user:

users.push({ name: 'name4', surn: 'surn4', });
