Passing a context as a parameter to the setInterval function

In fact, setInterval has a little-known syntax that solves the context problem described in the previous lesson.

The bottom line is that in addition to the first two parameters, additional parameters can be passed to setInterval. These additional parameters will go into the parameters of a bound function.

Let's look at an example. Let's say we have a function that takes two parameters like this:

function func(str1, str2) { console.log(str1, str2); }

Let's pass this function to setInterval as the first parameter, specify the time as the second parameter, and also make two additional parameters 'a' and 'b':

setInterval(func, 1000, 'a', 'b');

These additional parameters will automatically fall into the parameters of our func function.

Put it all together and run:

setInterval(func, 1000, 'a', 'b'); function func(str1, str2) { console.log(str1, str2); // will outputs 'a','b' every second }

Let's make our function anonymous (more compact but less obvious):

setInterval(function(str1, str2) { console.log(str1, str2); }, 1000, 'a', 'b');

We now solve the problem we have set using the described singularity of setInterval:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { setInterval(func, 1000, this); // passing this as a parameter function func(self) { // this gets into self console.log(self.value); } });

Let's make a more compact solution using an anonymous function:

let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { setInterval(function(self) { console.log(self.value); }, 1000, this); });

Let this code be given:

<input type="button" id="elem" value="10"> let elem = document.querySelector('#elem'); elem.addEventListener('click', function() { setInterval(function() { this.value--; }, 1000); });

The author of the code wanted that when a button is pressed, the value of this button decreases by 1 every second. However, when the button is pressed, nothing happens at all. Correct the mistake of the code author using the method learned in this lesson.
