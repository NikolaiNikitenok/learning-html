Working with HTML code of element in JavaScript

DOM elements have an innerHTML property that allows you to read the HTML code of those elements. Let's look at an example. Let's say we have the following tag:

<p id="elem"><b>text</b></p>

Get a reference to this tag into a variable:

let elem = document.querySelector('#elem');

Read the HTML code of the tag:

console.log(elem.innerHTML); // shows <b>text</b>

Change the tag text:

elem.textContent = '<i>!!!</i>';
