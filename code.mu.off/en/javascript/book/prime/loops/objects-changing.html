Changing objects with a loop in JavaScript

Objects can also be changed in a loop. Let, for example, we have such an object:

let obj = {a: 1, b: 2, c: 3};

Let's loop through it with for-in and increase each of its elements by 2 times:

for (let key in obj) { obj[key] = obj[key] * 2; }

Let's check the result:

console.log(obj);

let obj = {x: 1, y: 2, z: 3};
