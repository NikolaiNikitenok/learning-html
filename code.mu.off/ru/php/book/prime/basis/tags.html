Работа с HTML тегами в PHP

Теги HTML с точки зрения PHP представляют собой обычные строки. Давайте, например, с помощью тега <b> выведем на экран жирный текст:

<?php echo '<b>жирный</b>'; ?>

Можно использовать несколько echo:

<?php echo '<b>'; echo 'жирный'; echo '</b>'; ?>

Можно использовать конкатенацию:

<?php $str = 'жирный'; echo '<b>' . $str . '</b>'; ?>

Или вот так:

<?php $str = 'жирный'; $open = '<b>'; $close = '</b>'; echo $open . $str . $close; ?>

С помощью тега <i> выведите на экран курсивный текст.

С помощью тега <br> выведите на экран столбец чисел от 1 до 9.

Атрибуты тегов

Давайте с помощью тега <a> выведем на экран ссылку:

<?php echo '<a href="index.php">ссылка</a>'; ?>

Пусть теперь адрес и текст ссылки хранятся в отдельных переменных:

<?php $href = 'index.php'; $text = 'ссылка'; echo '<a href="' . $href . '">' . $text . '</a>'; ?>

С помощью тега <img> выведите на экран какую-нибудь картинку.

С помощью тега <input> выведите на экран инпут с каким-нибудь текстом.

С помощью тега <textarea> выведите на экран многострочное поле ввода с каким-нибудь текстом.
