Получение чисел Фибоначчи в PHP

В данном уроке мы с вами научимся получать числа Фибоначчи. Эти числа представляют собой последовательность, в которой первые два числа равны 0 и 1, а каждое последующее число равно сумме двух предыдущих чисел.

Вот решение описанной задачи:

<?php $one = 0; $two = 1; for ($i = 1; $i <= 10; $i++) { $current = $one + $two; $one = $two; $two = $current; echo $current . '<br>'; } ?>

Разберите мое решение описанной задачи, а затем самостоятельно, не подсматривая в мой код, решите эту задачу.

Решите задачу на нахождение ряда чисел, каждое из которых равно сумме трех предыдущих, а первые три числа равны 0, 1 и 2.
