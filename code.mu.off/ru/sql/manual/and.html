Команда AND

Команда AND является логическим 'И', которое позволяет комбинировать условия в команде WHERE.

Синтаксис

WHERE условие1 AND условие2

Пример

В данном примере из таблицы выбираются записи, у которых поле id больше 2-х и меньше 4-х:

SELECT * FROM employees WHERE id>=2 AND id<=4
