Инструкция break

Инструкция break осуществляет принудительный выход из цикла. Может завершать любые циклы: for, for-of, for-in, while.

Синтаксис

break;

Пример

Определим, есть ли в массиве число 3. Если есть - выведем в консоль слово 'есть' и выйдем из цикла:

let arr = [1, 2, 3, 4, 5]; for (let elem of arr) { if (elem == 3) { console.log('есть'); break; // выйдем из цикла } }
