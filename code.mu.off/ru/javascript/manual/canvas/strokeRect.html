Метод strokeRect

Метод strokeRect рисует в заданной точке контур прямоугольника. Первые два параметра задают координаты точки, в которой окажется верхний левый угол нарисованного прямоугольника.

Синтаксис

контекст.strokeRect(x, y, ширина, высота)

Пример

Давайте нарисуем прямоугольник с помощью strokeRect:

<canvas id="canvas" width="200" height="200" style="background: #f4f4f4;"></canvas> let canvas = document.querySelector('#canvas'); let ctx = canvas.getContext('2d'); ctx.strokeRect(50, 50, 100, 75);

:

Смотрите также

  • метод fillRect,
    который рисует закрашенный прямоугольник
  • метод rect,
    который рисует прямоугольник
  • метод lineTo,
    который рисует линии
  • свойство strokeStyle,
    которое задает цвет линии
