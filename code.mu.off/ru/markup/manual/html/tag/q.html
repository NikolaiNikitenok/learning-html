Тег q

Тег q предназначен для выделения коротких цитат в тексте. В атрибуте cite следует указывать ссылку на источник цитаты.

По умолчанию содержимое тега q отображается в кавычках. Вид кавычек зависит от языка текста, задаваемого атрибутом lang (если атрибута нет - браузер определит язык автоматически). Вид кавычек также можно поменять с помощью CSS свойства quotes.

Тег q можно использовать совместно с тегом cite, который нужен для указания ссылки на источник цитаты.

Атрибуты

Атрибут Описание
cite Ссылка на источник цитаты (при необходимости, если такой источник есть).

Пример

Давайте возьмем цитату в тег q и отобразим в кавычках:

<q>Все хорошо, что хорошо кончается</q>, сказал он.
