Свойство padding-left

Свойство padding-left устанавливает внутренний левый отступ элемента. Значением свойства служат любые единицы для размеров.

Синтаксис

селектор { padding: значение; }

Пример

Давайте сделаем внутренний левый отступ в 30px:

<div id="elem"> какой-то текст... </div> #elem { padding-left: 30px; width: 300px; border: 1px solid black; text-align: justify; }

:

Смотрите также

  • свойство padding-top,
    которое задает внутренний отступ сверху
  • свойство padding-bottom,
    которое задает внутренний отступ снизу
  • свойство padding-right,
    которое задает внутренний отступ справа
  • свойство padding,
    которое является сокращением для внутреннего отступа
  • свойство margin,
    которое задает внешний отступ
