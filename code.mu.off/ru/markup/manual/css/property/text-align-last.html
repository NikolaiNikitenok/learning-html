Свойство text-align-last

Свойство text-align-last устанавливает выравнивание последней строки текста: по левому краю, по правому, по центру, по левому и правому краям одновременно.

Обратите внимание на то, что если в элементе (например, в абзаце), только одна строка - то она считается одновременно первой и последней и выравнивание у нее будет такое, как задано в text-align-last, а не в text-align.

Синтаксис

селектор { text-align-last: center | left | right | justify | auto | start | end; }

Значения

Значение Описание
center По центру.
left По левому краю.
right По правому краю.
justify Одновременно и по правому краю, и по левому.
auto Не изменяет выравнивание элемента.
start Равно left, если текст идет слева направо и right, когда текст идет справа налево (арабский язык и т.п.).
end Равно right, если текст идет слева направо и left, когда текст идет справа налево.
inherit Наследует значение родителя.

Значение по умолчанию: auto.
