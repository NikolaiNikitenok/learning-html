Свойство text-transform

Свойство text-transform устанавливает регистр символов текста: верхний регистр (заглавные буквы), нижний регистр (строчные маленькие буквы), может делать заглавной первую букву каждого слова.

Синтаксис

селектор { text-transform: capitalize | lowercase | uppercase | none; }

Значения

Значение Описание
capitalize Первый символ каждого слова в предложении будет заглавным.
lowercase Все символы текста переводятся в нижний регистр.
uppercase Все символы текста переводятся в ВЕРХНИЙ РЕГИСТР.
none Не меняет регистр символов.

Значение по умолчанию: none.

Пример . Значение capitalize

Сейчас первая буква каждого слова станет заглавной:

<p> Lorem ipsum dolor sit amet. </p> p { text-transform: capitalize; }

:

Пример . Значение uppercase

Сейчас все символы станут заглавными:

<p> Lorem ipsum dolor sit amet. </p> p { text-transform: uppercase; }

:

Пример . Значение lowercase

Сейчас все символы станут строчными (то есть маленькими, не заглавными):

<p> LOREM IPSUM DOLOR SIT AMET. </p> p { text-transform: lowercase; }

:
